# es6-minesweeper
 Simple minesweeper game implementation with es6

# Running game

To run the game simple start a local server on the project main directory. You can use the **serve** npm package. Install it with: `npm i -g serve` and then just type `serve`.

![Simple game screenshot](https://user-images.githubusercontent.com/664712/60736151-43805b00-9f56-11e9-90c9-a391e152eee9.png)

# Running test

You need to run a local server too. Then navigate to: */tests/test.html* and do not forget to open the console in the Development Tools to watch the details.
